Flask-based To-Do Web Application built using Python and Flask. It allows users to add, view, update, and delete tasks for efficient daily task management. The project demonstrates CRUD operations, Flask routing, template rendering, and database integration with a simple and clean user interface.

Features

Add new tasks

Update existing tasks

Delete tasks

View all tasks

Simple and user-friendly UI

Tech Stack

Python

Flask

HTML, CSS

SQLite

How to Run

Clone the repository

Install required dependencies

Run python app.py

Open http://127.0.0.1:5000/ in your browser
